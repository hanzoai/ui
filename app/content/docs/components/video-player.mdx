## About

The Video Player component provides a full-featured video playback experience with:

- Custom control overlay with smooth animations
- Quality selection for multiple video sources
- Playback speed control (0.25x to 2x)
- Subtitle support with toggle
- Picture-in-picture mode
- Fullscreen support
- Comprehensive keyboard shortcuts
- Responsive design
- Progress bar with seeking
- Volume control with slider
- Auto-hiding controls during playback

## Usage

```tsx
import { VideoPlayer } from "@/registry/default/ui/video-player"

const videoSources = [
  {
    src: "/videos/sample-720p.mp4",
    type: "video/mp4",
    quality: "720p"
  },
  {
    src: "/videos/sample-1080p.mp4",
    type: "video/mp4",
    quality: "1080p"
  }
]

const subtitles = [
  {
    src: "/subtitles/english.vtt",
    label: "English",
    srcLang: "en",
    default: true
  },
  {
    src: "/subtitles/spanish.vtt",
    label: "Español",
    srcLang: "es"
  }
]

export function VideoPlayerDemo() {
  return (
    <VideoPlayer
      sources={videoSources}
      subtitles={subtitles}
      poster="/images/video-poster.jpg"
      size="lg"
      onPlay={() => console.log("Video started playing")}
      onPause={() => console.log("Video paused")}
      onTimeUpdate={(current, duration) => {
        console.log(`Progress: ${current}/${duration}`)
      }}
    />
  )
}
```

## Keyboard Shortcuts

The video player includes comprehensive keyboard shortcuts when focused:

| Key | Action |
|-----|---------|
| `Space` | Toggle play/pause |
| `F` | Toggle fullscreen |
| `M` | Toggle mute |
| `←` | Skip backward 10 seconds |
| `→` | Skip forward 10 seconds |
| `↑` | Increase volume |
| `↓` | Decrease volume |
| `C` | Toggle subtitles |
| `P` | Toggle picture-in-picture |

### Types

#### VideoSource

```tsx
interface VideoSource {
  src: string       // Video file URL
  type: string      // MIME type (e.g., "video/mp4")
  quality?: string  // Quality label (e.g., "720p", "1080p")
}
```

#### SubtitleTrack

```tsx
interface SubtitleTrack {
  src: string       // Subtitle file URL (WebVTT format)
  label: string     // Display name for the track
  srcLang: string   // Language code (e.g., "en", "es")
  default?: boolean // Whether this track is default
}
```

## Features

### Quality Selection
Provide multiple video sources with different qualities, and users can switch between them seamlessly:

```tsx
const sources = [
  { src: "/video-480p.mp4", type: "video/mp4", quality: "480p" },
  { src: "/video-720p.mp4", type: "video/mp4", quality: "720p" },
  { src: "/video-1080p.mp4", type: "video/mp4", quality: "1080p" },
]
```

### Subtitle Support
Add multiple subtitle tracks in WebVTT format:

```tsx
const subtitles = [
  { src: "/subs-en.vtt", label: "English", srcLang: "en", default: true },
  { src: "/subs-es.vtt", label: "Español", srcLang: "es" },
  { src: "/subs-fr.vtt", label: "Français", srcLang: "fr" },
]
```

### Picture-in-Picture
Enable picture-in-picture mode for multitasking:

```tsx
<VideoPlayer
  sources={sources}
  disablePictureInPicture={false} // Enable PiP
  onPlay={() => console.log("Playing in PiP")}
/>
```

### Responsive Design
The player adapts to different screen sizes and container dimensions:

```tsx
<VideoPlayer sources={sources} size="full" /> // Full width
<VideoPlayer sources={sources} size="sm" />   // Small size
```
