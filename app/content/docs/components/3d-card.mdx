A card component that responds to mouse movement with realistic 3D transformations, including perspective tilt, scaling, and optional glare effects. Supports touch devices and provides smooth animations.

## Usage

```tsx
import {
  Card3D,
  Card3DContent,
  Card3DDescription,
  Card3DFooter,
  Card3DHeader,
  Card3DTitle,
} from "@/components/ui/3d-card"
```

```tsx
<Card3D>
  <Card3DHeader>
    <Card3DTitle>3D Card Title</Card3DTitle>
    <Card3DDescription>This card responds to mouse movement</Card3DDescription>
  </Card3DHeader>
  <Card3DContent>
    <p>Interactive content with depth layers</p>
  </Card3DContent>
  <Card3DFooter>
    <p>Footer content</p>
  </Card3DFooter>
</Card3D>
```

```tsx
  </Card3DContent>
```
```tsx
</Card3D>
```

## API Reference

### Card3D

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `maxTilt` | `number` | `15` | Maximum tilt angle in degrees |
| `perspective` | `number` | `1000` | CSS perspective value in pixels |
| `scale` | `number` | `1.05` | Scale factor when hovered |
| `speed` | `number` | `400` | Animation duration in milliseconds |
| `glare` | `boolean` | `true` | Enable/disable glare effect |
| `glareMaxOpacity` | `number` | `0.7` | Maximum opacity of glare effect |

### Depth Components

All 3D card sub-components (`Card3DHeader`, `Card3DTitle`, `Card3DDescription`, `Card3DContent`, `Card3DFooter`) accept a `depth` prop:

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `depth` | `number` | varies | Z-axis translation in pixels for layered depth |

**Default depth values:**
- `Card3DTitle`: 30px
- `Card3DContent`: 25px
- `Card3DHeader`: 20px
- `Card3DFooter`: 15px
- `Card3DDescription`: 10px

## Features

- **Mouse Tracking**: Responds to cursor position with realistic tilt
- **Touch Support**: Works on mobile devices with touch events
- **Performance Optimized**: Uses GPU acceleration and requestAnimationFrame
- **Customizable**: Extensive props for fine-tuning effects
- **Accessible**: Maintains semantic HTML structure
- **Depth Layers**: Components can be positioned at different Z-depths
- **Smooth Animations**: Configurable transition timing
- **Glare Effect**: Optional realistic lighting overlay

## Technical Notes

- Uses CSS `transform3d` and `perspective` for hardware acceleration
- Implements proper touch event handling with `preventDefault`
- Calculates transforms based on mouse/touch position relative to card center
- Automatically resets transforms when mouse/touch leaves the card
- Uses `will-change: transform` for optimal performance

## Browser Support

Works in all modern browsers that support CSS 3D transforms:
- Chrome 36+
- Firefox 16+
- Safari 9+
- Edge 12+

## Accessibility

The 3D effects are purely visual enhancements and don't affect the underlying content structure. Users who prefer reduced motion can disable the effects through their system preferences, which will be respected by the component's CSS transitions.
