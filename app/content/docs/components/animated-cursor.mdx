## API Reference

### AnimatedCursor

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| isVisible | `boolean` | `true` | Whether the cursor is visible |
| size | `number` | `20` | Cursor size in pixels |
| color | `string` | `"#000000"` | Main cursor color |
| trailColor | `string` | `"#00000050"` | Trail cursor color |
| animationDuration | `number` | `200` | Animation duration in milliseconds |
| showTrail | `boolean` | `true` | Whether to show trail effect |
| trailLength | `number` | `8` | Number of trail elements |
| blendMode | `BlendMode` | `"normal"` | Blend mode for cursor |
| hoverScale | `number` | `1.5` | Scale factor on hover |
| hideOnTouch | `boolean` | `true` | Whether to hide on touch devices |
| className | `string` | - | Custom CSS class |
| clickAnimationDuration | `number` | `100` | Click animation duration |
| zIndex | `number` | `9999` | Z-index for cursor |

type BlendMode =

## Features

### Automatic Hover Detection

The cursor automatically detects interactive elements and changes appearance:

- **Buttons and links**: Pointer cursor with scale animation
- **Text inputs**: Text cursor (line shape)
- **Draggable elements**: Grab cursor
- **Custom elements**: Use `data-cursor` attribute

```tsx
// Custom cursor types
<div data-cursor="pointer">Custom pointer</div>
<div data-cursor="text">Custom text</div>
<div data-cursor="grab">Custom grab</div>
<div data-cursor="grabbing">Custom grabbing</div>
```

### Touch Device Support

The cursor automatically hides on touch devices by default. You can control this behavior with the `hideOnTouch` prop.

### Performance Optimized

- Uses `requestAnimationFrame` for smooth animations
- Efficient trail rendering
- Minimal re-renders with optimized state management

### Accessibility

The component preserves the original cursor behavior and doesn't interfere with keyboard navigation or screen readers.

## Advanced Usage

### Custom Cursor Styles

Create different cursor styles for different sections:

```tsx
function App() {
  const [cursorStyle, setCursorStyle] = useState('default')

  return (
    <div>
      <AnimatedCursor
        color={cursorStyle === 'dark' ? '#ffffff' : '#000000'}
        blendMode={cursorStyle === 'blend' ? 'difference' : 'normal'}
      />

      <section
        onMouseEnter={() => setCursorStyle('dark')}
        onMouseLeave={() => setCursorStyle('default')}
        className="bg-black text-white p-8"
      >
        Dark section
      </section>

      <section
        onMouseEnter={() => setCursorStyle('blend')}
        onMouseLeave={() => setCursorStyle('default')}
        className="p-8"
      >
        Blend mode section
      </section>
    </div>
  )
}
```

### Conditional Rendering

Show different cursors based on application state:

```tsx
function App() {
  const [isLoading, setIsLoading] = useState(false)

  return (
    <div>
      <AnimatedCursor
        color={isLoading ? '#ff6b35' : '#000000'}
        showTrail={!isLoading}
        animationDuration={isLoading ? 500 : 200}
      />

      <button onClick={() => setIsLoading(!isLoading)}>
        Toggle Loading
      </button>
    </div>
  )
}
```

## Troubleshooting

### Cursor Not Appearing

1. Check if `isVisible` is set to `true`
2. Ensure the component is mounted on the client side
3. Verify that touch device detection is working correctly

### Performance Issues

1. Reduce `trailLength` for better performance
2. Disable trail with `showTrail={false}` if not needed
3. Increase `animationDuration` to reduce update frequency

### Z-Index Issues

If the cursor appears behind other elements, increase the `zIndex` prop:

```tsx
<AnimatedCursor zIndex={99999} />
```
