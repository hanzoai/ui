## Usage

```tsx
import { AICode } from "@/components/ui/ai-code"

export function CodeAssistantDemo() {
  const handleGenerate = async (prompt: string) => {
    // Your AI code generation logic
    const response = await fetch('/api/generate-code', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ prompt })
    })
    const { code } = await response.json()
    return code
  }

  const handleExplain = async (code: string, line?: number) => {
    // Your code explanation logic
    const response = await fetch('/api/explain-code', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ code, line })
    })
    const { explanations } = await response.json()
    return explanations
  }

  const handleRefactor = async (code: string) => {
    // Your code refactoring logic
    const response = await fetch('/api/refactor-code', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ code })
    })
    const { refactoredCode } = await response.json()
    return refactoredCode
  }

  return (
    <AICode
      defaultValue="// Start typing or ask AI to generate code..."
      language="typescript"
      height="600px"
      onGenerate={handleGenerate}
      onExplain={handleExplain}
      onRefactor={handleRefactor}
      features={['generate', 'explain', 'refactor', 'debug', 'test', 'document', 'review']}
      placeholder="Ask AI to generate code, explain functions, or help with debugging..."
    />
  )
}
```

## Features

The AI Code component provides comprehensive AI-powered development assistance:

### Core Features
- **Code Generation**: Natural language to code conversion
- **Code Explanation**: Inline annotations and hover explanations
- **Refactoring**: Automated code improvement suggestions
- **Debugging**: Bug detection and fix recommendations
- **Test Generation**: Automated test case creation
- **Documentation**: Auto-generated code documentation
- **Code Review**: Comprehensive code analysis and suggestions

### Advanced Features
- **Security Scanning**: Vulnerability detection and fixes
- **Multi-file Support**: Work with multiple files simultaneously
- **Language Detection**: Automatic programming language detection
- **Import Resolution**: Smart import and dependency management
- **Performance Analysis**: Code performance suggestions
- **Framework Detection**: Context-aware framework suggestions

### Types

#### CodeFeature
```tsx
type CodeFeature =
  | 'generate'
  | 'explain'
  | 'refactor'
  | 'debug'
  | 'test'
  | 'document'
  | 'review'
```

#### Explanation
```tsx
interface Explanation {
  id: string
  line: number
  content: string
  type: 'explanation' | 'suggestion' | 'warning' | 'error'
}
```

#### CodeSuggestion
```tsx
interface CodeSuggestion {
  id: string
  type: CodeFeature
  title: string
  description: string
  code?: string
  startLine?: number
  endLine?: number
  severity?: 'info' | 'warning' | 'error'
}
```

#### SecurityIssue
```tsx
interface SecurityIssue {
  id: string
  line: number
  severity: 'low' | 'medium' | 'high' | 'critical'
  message: string
  fix?: string
}
```

## Integration Guide

### Backend API Integration

Implement the following API endpoints to connect with your AI backend:

```typescript
// /api/generate-code
export async function POST(request: Request) {
  const { prompt } = await request.json()

  // Call your AI service (OpenAI, Claude, etc.)
  const code = await generateCode(prompt)

  return Response.json({ code })
}

// /api/explain-code
export async function POST(request: Request) {
  const { code, line } = await request.json()

  const explanations = await explainCode(code, line)

  return Response.json({ explanations })
}

// /api/security-scan
export async function POST(request: Request) {
  const { code } = await request.json()

  const issues = await scanForSecurityIssues(code)

  return Response.json({ issues })
}
```

### Custom AI Providers

Integrate with different AI providers:

```typescript
// OpenAI integration
const openaiProvider = {
  async generateCode(prompt: string) {
    const response = await openai.chat.completions.create({
      model: "gpt-4",
      messages: [{ role: "user", content: `Generate code: ${prompt}` }]
    })
    return response.choices[0].message.content
  }
}

// Claude integration
const claudeProvider = {
  async generateCode(prompt: string) {
    const response = await anthropic.messages.create({
      model: "claude-3-sonnet-20240229",
      messages: [{ role: "user", content: `Generate code: ${prompt}` }]
    })
    return response.content[0].text
  }
}
```

## Best Practices

### Security Considerations
- Always validate and sanitize AI-generated code before execution
- Implement rate limiting for AI API calls
- Use proper authentication for AI endpoints
- Review security scan results before applying fixes

### Performance Optimization
- Implement debouncing for real-time AI features
- Cache common AI responses
- Use progressive loading for large codebases
- Optimize Monaco Editor configuration for your use case

### User Experience
- Provide clear loading states during AI operations
- Implement proper error handling and fallbacks
- Allow users to customize AI feature availability
- Provide keyboard shortcuts for common actions

### Accessibility
- Ensure proper ARIA labels for all interactive elements
- Support screen readers with meaningful descriptions
- Provide keyboard navigation for all features
- Implement proper focus management

## Troubleshooting

### Common Issues

**Monaco Editor not loading**
- Ensure Monaco Editor assets are properly loaded
- Check for conflicting CSS or JavaScript
- Verify import paths are correct

**AI features not working**
- Check API endpoint implementations
- Verify authentication tokens
- Review network requests in browser dev tools

**Performance issues**
- Reduce editor features for large files
- Implement virtual scrolling for file lists
- Use Web Workers for heavy computations

**Theme not applying**
- Verify theme provider is properly configured
- Check CSS variable definitions
- Ensure theme prop is correctly passed
