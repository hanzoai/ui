## Available Tools

### Code Generation

AI-powered code generation and completion tools.

```tsx
import { AICodeGenerator } from "@hanzo/ui/ai"

export default function CodePage() {
  return (
    <AICodeGenerator
      language="typescript"
      context="React component"
      onGenerate={(code) => console.log(code)}
    />
  )
}
```

### Content Analysis

Analyze and process text content with AI.

```tsx
import { AIContentAnalyzer } from "@hanzo/ui/ai"

export default function AnalysisPage() {
  return <AIContentAnalyzer onAnalyze={(result) => console.log(result)} />
}
```

### Smart Search

AI-enhanced search with semantic understanding.

```tsx
import { AISearch } from "@hanzo/ui/ai"

export default function SearchPage() {
  return (
    <AISearch
      placeholder="Ask anything..."
      onSearch={(query) => console.log(query)}
    />
  )
}
```

## Features

- **Code Generation**: Generate code from natural language
- **Content Analysis**: Sentiment, keywords, summarization
- **Smart Search**: Semantic search with AI
- **Auto-completion**: Intelligent suggestions
- **Translation**: Multi-language support
- **Classification**: Content categorization

## Integration

AI Tools integrate with popular AI providers:

- Anthropic (Claude)
- Google (Gemini)
